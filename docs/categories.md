# Категории приложений

Для правильной сортировки приложений в клиенте MEmarket используется файл `metadata.yaml`. В нем вы можете указать параметр `category:`, чтобы приложение попало в нужный раздел магазина.

Если параметр `category:` не указан в файле, написано с ошибкой или написано нестандартно, приложение автоматически попадет в общую категорию **"Прочее"**.

---

## Список актуальных сортируемых категорий:

| category: "..." | Название |
|---|---|
| "games" | Игры |
| "store" | Др. магазины приложений |
| "entertaintment" | Развлечения |
| "music+video" | Музыка + видео |
| "tools" | Инструменты |
| "livestyle" | Лайвстайл |
| "news+weather" | Новости + погода |
| "health+fitness" | Здоровье + фитнес |
| "photo" | Фото |
| "social" | Социальные |
| "sports" | Спорт |
| "business" | Бизнес |
| "education" | Обучение |


**Пример правильного оформления в metadata.yaml:**

```yaml
title: "Telegram"
creator: "Telegram Messenger LLP"
summary: "Официальный клиент Telegram"
category: "social"
```

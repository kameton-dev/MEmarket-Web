# Начало

Millennium Market состоит из официального репозитория проверенных, необходимых приложений и набора инструментов для их удобной настройки, поддержки и комфортного использования. Этот проект создан и развивается силами независимого энтузиаста при поддержке небольшого, но преданного своему делу сообщества. MEmarket доказывает, что кастомные каталоги и забота о Windows Phone — это история не только «для гиков». Проект стремится быть максимально простым, понятным и дружелюбным к обычным пользователям.

Проект ставит своей целью вдохнуть новую жизнь в устройства Windows Phone, предоставляя независимую, открытую и гибкую альтернативу другим неофициальным магазинам приложений. В состав MEmarket входит как клиентское приложение для мобильных устройств, так и серверные утилиты для автоматизации созданий альтернативных репозиторий и сборки пакетов.

---

## Архитектура

Проект MEmarket разделен на два ключевых компонента, которые работают в связке для обеспечения полной независимости экосистемы:

1. **Клиенты MEmarket** — приложения для Windows Phone 8.1 и Windows 10 Mobile, позволяющие пользователям подключать сторонние репозитории, искать, скачивать и устанавливать приложения прямо на устройство.
2. **Инструменты репозитория (`memarketrepo`)** — консольная утилита на Python для автоматизации создания, валидации и обновления JSON-индексов ваших собственных репозиториев.

---

## Утилиты управления репозиторием

"Зазеркалье" MEmarket содержит специализированные сценарии для генерации метаданных и управления каталогом приложений:

* **Создание собственных репозиториев:** Вы можете развернуть зеркало или полностью независимый альтернативный магазин для публикации своих XAP/APPX пакетов.
* **Автоматизация и валидация:** Инструменты автоматически сканируют папки с приложениями, проверяют JSON-схему (включая минимальные сборки ОС и цифровые сертификаты приложений) и собирают единый файл индекса.
* **Удобность:** Утилита `memarketrepo` доступна для Windows (пока что), что позволяет развернуть центр управления ваших репозиторий на вашем ПК.

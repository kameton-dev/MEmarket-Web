# Скачивание приложений с клиентов MEmarket

В зависимости от клиента, приложения скачиваются по разному:

---

## MEmarket Silverlight

Приложение можно скачать, нажав на кнопку внизу экрана и выбрав место сохранения пакета.

> **ВНИМАНИЕ!**
> Данный клиент, к сожалению, пока не поддерживает функцию развертывания. Это значит что дальнейшая установка пакета в телефон осуществляетсячерез CMD Injector или через пк с утилитой Windows Phone Application Deployment Tool 8.1

---

## MEmarket UWP

Приложение можно скачать, нажав на кнопку "Скачать". Потом, благодаря установленному пакету `Microsoft.App.Installer.v2019.1019.1.0.appxbundle`, после скачивания в диалоговом окне вы можете выбрать **"Установщик приложений"** и установить скачанный пакет на телефон.

> **Важное уточнение!**
> Установщик приложений не поддерживает установку XAP пакетов. В клиенте это предусмотренно, и припопытке скачивания может вылезти диалоговое окно с предложением сохранить файл на устройство, для дальнейшего развертывания в CMD Injector / WPADT8.1

# Установка клиентов MEmarket

Для доступа к магазину вам необходимо установить клиентское приложение на ваше мобильное устройство. Выберите инструкцию ниже в зависимости от версии вашей операционной системы.

---

## 1. MEmarket Silverlight (для Windows Phone 8.1)

Этот клиент оптимизирован для работы на классической ОС Windows Phone 8.1.

### Подготовка
* Ваш смартфон должен быть разблокирован для установки сторонних приложений (разблокирован загрузчик и выполнен **Interop Unlock**).
* Вам понадобится ПК под управлением Windows и USB-кабель.

### Пошаговая инструкция
1. Скачайте актуальный файл `MEmarket_Release_ARM.xap` из [раздела релизов](../download/index.html).
2. Подключите смартфон к компьютеру через USB.
3. Откройте утилиту для развертывания приложений **Windows Phone Application Deployment 8.1**.
4. Выберите скачанный файл и нажмите кнопку **Развернуть (Deploy)**.
5. После успешного завершения MEmarket Silverlight появится в списке приложений на вашем телефоне.

## 2. MEmarket UWP (для Windows 10 Mobile)

Современный клиент, разработанный на базе Universal Windows Platform для устройств под управлением Windows 10 Mobile.

### Подготовка
1. На телефоне перейдите в **Параметры** -> **Обновление и безопасность** -> **Для разработчиков**.
2. Активируйте пункт **Режим разработчика** и подтвердите действие.
3. Обязательно скачайте дополнительные компоненты из [раздела релизов](../download/index.html).

### Пошаговая инструкция
1. Скачайте пакет приложения `MEmarket-UWP_x.x.x.x_ARM.appx`, универсальный установщик `Microsoft.App.Installer.v2019.1019.1.0.appxbundle` и его зависимости (`Microsoft_VCLibs_140_00_UWPDesktop_14_0_27629_0_arm_8wekyb3d8bbwe` и  `Microsoft.VCLibs.140.00_14.0.27323.0_arm__8wekyb3d8bbwe`) прямо на устройство через браузер или скиньте его в память телефона.
2. Откройте стандартное приложение **Проводник**, найдите скачанные файлы и установите их строго по порядку:
- `Microsoft_VCLibs_140_00_UWPDesktop_14_0_27629_0_arm_8wekyb3d8bbwe.appx`;
- `Microsoft.VCLibs.140.00_14.0.27323.0_arm__8wekyb3d8bbwe.appx`;
- `Microsoft.App.Installer.v2019.1019.1.0.appxbundle`;
- `MEmarket-UWP_x.x.x.x_ARM.appx`.

Весь процесс установки происходит в фоновом режиме и занимает около минуты. Приложение MEmarket появится в общем списке программ.

## Возникли проблемы?
* **Ошибка установки на WP8.1:** Убедитесь, что устройство полностью разблокировано и на нем установлены правильные твики реестра для обхода ограничений старого Marketplace.
* **Приложение на W10M не появляется:** Проверьте, включен ли режим разработчика. Без него система полностью блокирует установку любых `.appx` файлов в обход встроенного магазина Microsoft. Если проблема осталась, вы можете попробовать установить пакеты через **Портал устройств**.
